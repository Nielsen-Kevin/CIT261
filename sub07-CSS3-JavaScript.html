Style display

In the hangeman app the stick figure is made of embedded SVG tags so I can interactwith it like HTML. The tags are forced hidden and as you play, the tag are reveals one by one creating the hanging stick figure.

// Hide Man for(let i = 0; i < man.length; i++) { document.getElementById(man[i]).style.display = "none"; } // Wrong letter guessed wrong_count += 1; document.getElementById(man[wrong_count-1]).style.display = "block";

Style color, Style top

The message in the game changes color base on if your guess is wrong or right. The message also looked to static so I made it move to give it some life.

function message(status) { // Setup messages from array random_number = Math.round(Math.random() * (messages[status].length - 1)); document.getElementById('message').innerHTML = messages[status][random_number]; document.getElementById('message').style.color = txtColor[status]; // Gives the message some life let pos = -20; let id = setInterval(frame, 5); function frame() { if (pos == 0) { clearInterval(id); } else { pos++; document.getElementById("message").style.top = pos + "px"; } } }

No stick figure were harmed in the making of this game.

Class Manipulating

When I first read this assignment I didn't catch the "class" part I only thought of CSS manipulating with JavaScript. Note that classes are part of CSS but you don't only use classes strictly for CSS in the html tag. It is just a property. If your class has no CSS styles in it, I would argue that this method is not true CSS manipulation, it is only changing the class. Changing the style directly in JavaScript is true CSS manipulation, what we did above.

Now the biggest reason to use classes in JavaScript is the huge customization potential. You can style your class with out having to go back to your JavaScript directly modifying the CSS every time you reuse your code for other projects.

let thisDay = document.createElement('div'); thisDay.setAttribute('id', this.key); thisDay.setAttribute('class', 'day'); // Is it today if(calendar[this.key].year == today.getFullYear() && calendar[this.key].month == today.getMonth()+1 && calendar[this.key].day == today.getDate()) { thisDay.classList.add('today'); this.today = true; }
