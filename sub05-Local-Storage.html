Local Storage

Local Storage API, Storing and Retrieving Simple Data, Arrays, Associative Arrays, and Objects

Different Names

localStorage, DOM storage or web storage. All these names are in use, and all reference the same thing.

Properties

localStorage.length //Read only

Returns an integer representing the number of data items stored in the Storage object.

Methods

localStorage.key()

When passed a number n, this method will return the name of the nth key in the storage.

// retrieve a value value = localStorage.getItem( 'key' );

When passed a key name, will return that key's value.

// store a value localStorage.setItem( 'key', 'value' );

When passed a key name and value, will add that key or update that key's value if it already exists.

// remove the value localStorage.removeItem( 'key' );

When passed a key name, will remove that key from the storage.

localStorage.clear()

When invoked, will empty all keys out of the storage.

Code Example

// only string values can be stored so for objects, use JSON var objArray = { title: 'Cookies are old', author: 'Gordan' } localStorage.setItem( 'objArray', JSON.stringify( objArray ) ); // and to retrieve var objArray = JSON.parse( localStorage.getItem( 'objArray' ) );
